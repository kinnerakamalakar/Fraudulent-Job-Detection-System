# Fraudulent Job Detection System 🚀

## 📌 Overview
This project detects fraudulent job postings using Machine Learning and Flask web application.

It analyzes job-related features and predicts whether a job is **REAL or FAKE**.

---

## ⚙️ Technologies Used
- Python 🐍  
- Flask 🌐  
- Machine Learning 🤖  
- Scikit-learn 📊  
- Pandas & NumPy  

---

## 📂 Project Structure
- app.py → Flask web app  
- train_model.py → ML model training  
- predict.py → Prediction logic  
- dataset.csv → Training dataset  
- templates/ → HTML pages  
- static/ → CSS files  

---

## 🚀 How to Run Project

```bash
pip install -r requirements.txt
python train_model.py
python app.py
